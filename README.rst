========================
A simple premutation-based wordlist generator

Example
-------

* input.txt

::

  github admin


* output.txt

::

  ...
  g!thub_@dMiN
  ...
  Github@admin
  ...
  Github@adm!n
  ...
  GitHub$Adm!n
  ...
  917HU8@DM1N



Run
---

.. code:: shell

  ./pw.gen input.txt output.txt
